좋아, 그러면 지금은 **코드보다 구조와 기능 중심 설계**를 잡는 단계로 가보자.
목표는 **누구나 블록 클릭만으로 화면 만들기 가능 + 독립적 컴포넌트 끼우기 가능** 수준으로 포폴용 설계.

---

## 1️⃣ 폴더 구조 예시

```
/src
  /components          <-- 블록 단위 컴포넌트
    Header.ts
    Hero.ts
    Text.ts
    Image.ts
    Form.ts
    Footer.ts
    Modal.ts
  /pages               <-- 페이지 단위 컴포넌트 (조립용)
    Home.ts
    Posts.ts
    Editor.ts
  /state               <-- 상태 + Undo/Redo
    store.ts
    types.ts
  /utils               <-- 유틸 함수
    dom.ts
    helpers.ts
  main.ts              <-- 진입점, 툴바/렌더 초기화
  style.css            <-- 최소 CSS
/assets
  /images
```

* **components/** → 슬롯 단위 블록 (Header, Hero, Text, Image, Form, Footer, Modal)
* **pages/** → 블록을 조립해서 페이지 단위로 보여주는 컴포넌트
* **state/** → 전체 페이지 상태 관리, Undo/Redo, 선택 상태
* **utils/** → DOM 조작, UUID 생성 등 헬퍼
* **main.ts** → 툴바, 페이지 렌더링, 이벤트 연결

## 2️⃣ 핵심 기능별 동작 흐름

### 2-1. 페이지 상태

* `PageState` → 현재 화면 구성, 선택 블록, undo/redo stack
* `sections: Section[]` → 순서대로 렌더
* `selectedId: string` → 선택 블록
* `undoStack / redoStack` → 상태 기록

---

### 2-2. 렌더링 흐름

1. `renderPage(state, root)` → sections 배열 순서대로 DOM 생성
2. 각 블록은 **독립적 컴포넌트** → mount/update 가능
3. 클릭하면 `selectedId` 업데이트 → 선택 블록 강조
4. 선택 블록은 **편집 옵션** 표시 (텍스트/이미지/폼 필드)

---

### 2-3. 블록 조작

* **추가:** 툴바 버튼 클릭 → 새 Section push → renderPage 호출
* **삭제:** 선택 블록 삭제 → sections 배열에서 제거 → renderPage 호출
* **순서 변경:** 선택 블록 위/아래 버튼 → 배열 순서 변경 → renderPage 호출
* **편집:** 선택 블록 데이터 수정 → update() → renderPage 호출

> 핵심: 블록 단위로 독립적 컴포넌트, 서로 상태 의존 최소화

---

### 2-4. Undo / Redo

* **pushUndo()** → sections deep copy → undoStack에 저장
* **undo() / redo()** → stack pop/push → sections 교체 → renderPage 호출

---

### 2-5. 저장 / 불러오기

* **Save:** localStorage.setItem("page", JSON.stringify(sections))
* **Load:** localStorage.getItem("page") → sections → renderPage

---

### 2-6. 독립 컴포넌트 끼우기

* 모든 슬롯은 **독립적 mount/update 함수** 제공
* 페이지 조립 시, 배열 순서대로 끼우면 됨
* 새로운 블록 추가도 **components/** 폴더에 파일 하나 추가 + factory 함수만 구현하면 OK

> 장점: 초보 사용자도 클릭/드래그/선택/삭제로 화면 조립 가능,
> 개발자는 컴포넌트 단위로 기능 추가 가능

---

## 3️⃣ 핵심 UX 포인트

1. **툴바** → 블록 추가 / Undo / Redo / Save
2. **미리보기 영역** → 선택 블록 강조
3. **선택 블록 옵션 패널** → 텍스트 변경, 이미지 선택, 폼 필드 편집
4. **Drag & Drop** (선택 사항) → 배열 순서 변경 가능
